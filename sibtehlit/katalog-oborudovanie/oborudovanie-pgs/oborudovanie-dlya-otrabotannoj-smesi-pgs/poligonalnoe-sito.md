# Полигональное сито

**Полигональное барабанное сито** служит для просеивания свежего песка, а также отработанных формовочных смесей.

Для современных участков песчано-глинистой формовки требуется смесь с постоянными и однородными качественными характеристиками, которые могут быть получены только при наличии хорошего охладителя, эффективного смесителя и полигонального сита. Полигональное сито используется на участке смесеподготовки для обеспечения хорошей очистки формовочной смеси путем удаления и измельчения комьев смеси, удаления остатков стержневой смеси. Для достижения более высокой эффективности это устройство должно быть установлено надлежащим образом с соблюдением требуемых параметров по вытяжке, а его местоположение в существующем комплексе должно быть тщательно подобрано.

Сито представляет собой усеченный барабан с шестью гранями. Привод барабана осуществляется через редуктор

Подлежащий просеиванию сыпучий материал посту­пает во внутрь вращающегося барабана \(через его малый торец\). Внутри барабана материал пересыпается с грани на грань. При этом мелкие частицы просеиваются и падают вниз. Часть крупных комьев разбивается и просеивается. Уцелевшие комья скатываются к большому торцу барабана и выпадают из барабана в специальный бункер.

Размер ячейки нержавеющей сетки может быть подобран в соответствии с фракцией песка \(согласовывается с заказчиком\). Эффективно удаляются включения 4-5мм и боле крупные.
