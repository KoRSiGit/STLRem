# Воздушный охладитель

**Воздушный охладитель** предназначен для охлаждения отработанной смеси в кипящем слое. Устанавливается после полигонального сита на участке смесеподготовки. Горячая смесь загружается в установку сверху. В нижнюю часть воздуходувкой нагнетается избыточное давление для создания псевдокипящего слоя. Два вибродвигателя обеспечивают направленное просеивание и транспортировку отработанной смеси. Тепрература смеси определяется при помощи ифракрасного пирометра. При привышении температуры смеси выше установленного значения, внутри охладителя распыляется вода через ряры форсунок.

Воздушный охладитель поставляется в комплекте с воздуходувкой. Для качественной работы охладитель должен быть обеспечен системой пылеулавливания.

Воздушный охладитель может быть встроен в существующую систему смесеприготовления с привязкой к системе пылеулавливания и привязкой к системе управления автоматической формовочной линии.
