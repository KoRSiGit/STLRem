# Пескострельный стержневой автомат

Пескострельный стержневой автомат предназначен для автоматизированного производства песчано-смоляных стержней как по горячим таки и по холодным ящикам. Автомат устанавливается в непосредственной близости формовочной линии. Принцип работы: отъемные элементы стержневого ящика в исходном положении открыты, они закрываются при помощи гидравлических приводов, стержневая смесь со сжатым воздухом вдувается в стержневой ящик, далее в зависимости от технологии ящик нагревается за счет электрического сопротивления \(горячий процесс\), либо через ящик пропускается газообразный катализатор \(холодный процесс\). Смесь затвердевает в рабочей полости ящика образуя твердый стержень. После раскрытия ящика стержень извлекается при помощи толкателей.

Пескострельные стержневые автоматы разделяют по технологии раскрытия стержневого ящика в зависимости от сложности конфигурации стержня. Различают автоматы с вертикальным, горизонтальным или комбинированным разъемом стержневого ящика.

Для изготовления стержней по технологии COLD-BOX-AMIN стержневые автоматы оснащаются современными системами газогенерации и нейтрализации отходящих газов, а сам автомат при этом располагается в герметичной кабине, для предупреждения воздействия паров амина на оператора установки.

Технология изготовления стержня по CO2-процессу предполагает дополнительную установку газогенератора для углекислоты.
