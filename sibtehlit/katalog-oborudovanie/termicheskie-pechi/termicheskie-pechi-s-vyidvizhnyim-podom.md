# Термические печи с выдвижным подом

**Термические печи с выдвижным \(выкатным\) подом** предназначены для термической обработки изделий из черных и цветных сплавов. Принцип работы термопечи заключается в следующем: металлические изделия, подлежащие термической обработке, размещают на выдвижном поде печи. Максимальная масса изделий, загружаемых в печь \(масса садки\), ограничена грузоподъемностью пода и мощностью термической печи. Далее загруженный под заводится в камеру печи, после чего дверца печи закрывается. Запуск нагревательных элементов осуществляется только при закрытой дверце. Скорость набора температуры в камере термической печи, выдержка при установленной температуре осуществляется в автоматическом режиме по заданной программе. Контроль температуры в камере осуществляют термопары.

Термопечь состоит из:
- Сварного корпуса, внутренняя часть которого выполняется из высокоэффективного огнеупорного волокна;
- Нагревательных элементов в виде ленты высокого сопротивления из материала суперфехраль;
- Дверцы печи;
- Выдвижного \(выкатного\) пода,выложенного жаропрочным кирпичом;
- Шкафа электроконтроля.

По желанию Заказчика печь может быть поставлена с **отличным от стандартных** размером рабочей камеры, а также быть оснащена дополнительным оборудованием.
