# Пресс-формы для ЛПД

**Пресс-формы** для изготовления металлических отливок **под высоким давлением \(ЛПД\)** поставляются нашей компанией для машин с горячей и **[холодной камерой прессования](mashinyi-dlya-litya-pod-vyisokim-davleniem-s-xolodnoj-kameroj-pressovaniya.md "Машина для литья под давлением")**. Принимаем заказы на пресс-формы любой сложности. Изготовление пресс-форм в минимальные сроки. Каждая пресс-форма проходит обязательные «горячие» испытания с получением годных отливок. Отливки отправляются в адрес клиента для проверки и подтверждения геометрических размеров и качества отливок. Только после этого пресс-форма отправляется Заказчику с завода. При разработке проекта пресс-форм используется современное программное обеспечение, позволяющее правильно рассчитать конфигурацию литниковой системы, линии разъема и уклоны. Материал пресс-форм – сталь Н13 \(американский стандарт, отечественный аналог - 4Х5МФ1С \(другое обозначение ЭП572\)- Сталь инструментальная штамповая\) **с ХТО.**Гарантия на пресс-формы предоставляется в зависимости от типа сплава: для алюминиевых сплавов - 100 000 запрессовок, для цинковых - до 500 000 запрессовок.

Для оценки стоимости пресс-формы необходимо предоставить 2D чертеж или 3D модель отливки, либо фотографию отливки с указанием габаритных размеров, марку сплава, усилия запирания и прессования существующей машины литья под давлением.
