# Пресс-формы ЛГМ

**Пресс-формы ЛГМ**, поставляемые нашей компанией предлагаются как для ручного изготовления моделей в автоклавах, так и для модельных автоматов. Разработка проекта пресс-формы ЛГМ осуществляется специализированным конструкторским отделом. При выполнении проекта учитываются индивидуальные параметры полистирольных моделей: толщина стенки, место задувки гранул полистирола, венты – их общая площадь и месторасположение, подвод пара к пресс-форме, способ охлаждения. Для сложных моделей, которые затруднительно получить в одной форме предлагается комплект пресс-форм более простой конфигурации, с последующей сборкой в сложную модель.

Для изготовления пресс-форм используется материал, обладающей коррозионной стойкостью, высокой теплопроводностью, высокими механическими свойствами \(алюминиевые сплавы Д16, АЛ19, АЛ24, латуни\).

Каждая пресс-форма подвергается обязательным испытаниям с получением годных полистирольных моделей. Модели проверяются: на соответствие геометрическим размерам, качество поверхности, сплошность по объему. Если по каким-то параметрам годная модель не получается, то происходит доработка пресс-формы ЛГМ.

Готовая пресс-форма ЛГМ передаётся заказчику с комплектом годных моделей и рекомендациями по изготовлению этих моделей.
