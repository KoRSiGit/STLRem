# Оснастка для центробежного литья

Оснастка для центробежных машин выполняется из качественных кованных заготовок. Приоритетной маркой стали для оснастки центробежных машин является легированная сталь 42CrMo либо 42 CrNiMo. Стойкость оснастки до 700 циклов \(при условии применения разделительного покрытия на рабочей поверхности\).
