# Передаточная тележка

**Передаточная тележка** используются в формовочных линиях ХТС принципа FAST-LOOP для перемещения подмодельных плит и полуформ, кроме этого передаточные тележки могут использоваться на заливочных ветвях. Для перемещения подмодельных плит и опочных полуформ передаточные тележки оснащаются рольганговым транспортером, для перемещения безопочных полуформ применяется ленточный транспортер, для перемещения форм на заливочных ветвях применяется тележка с гидравлическим толкателем. Длина и ширина транспортеров подбираетя в зависимости от максимальных размеров формы. Для позиционирования подмодельных плит используются бесконтактные концевые выключатели. Приводы транспортеров, а также приводы перемещения тележки оснащаются частотными преобразователями частоты для регулирования скорости. Система управления передаточными тележками интегрируется в систему управления линией ХТС. Для небольших производств возможна поставка неприводных передаточных тележек.
