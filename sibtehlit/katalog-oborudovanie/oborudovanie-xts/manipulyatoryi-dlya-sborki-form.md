# Манипулятор для сборки форм

**Подвесной манипулятор** применяется для транспортировки, кантования и сборки форм из песчаных смесей. Устройство может использоваться как для форм в опоках так и безопочных форм с применением специализированных цапф. Плавность устройства кантования гарантирует переворачивание формы без ее повреждения. Манипулятор может поставляться в 3-х исполнениях:
- C цепными стропами;
- С текстильными стропами.
- Зажимной манипулятор \(гидравлический или пневматический\)

Основные части устройства:
- Рама манипулятора;
- Балансирующее устройство;
- Устройство для регулирования расстояния между стропами;
- Кантующее устройство.

Рама выполнена сварной из листовой и профильной стали. Когда манипулятор долго не используется его устанавливают на специальные стойки. Балансирующее устройство представляет собой механизм, который проводит корректировку центра тяжести при при его смещении под действием нагрузки. За счет такой корректировки обеспечивается стабильная и безопасная работа при подъеме, перемещении и кантовке груза. Устройство регулирования расстояния между стропами позволяет подстраивать манипулятор практически под любые задачи и грузы. За счет кантующего устройства происходит переворот груза.
