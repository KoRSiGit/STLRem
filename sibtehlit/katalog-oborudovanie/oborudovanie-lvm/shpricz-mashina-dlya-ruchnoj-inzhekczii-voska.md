# Шприц-машина для ручной инжекции воска

**Шприц-машина** **для ручной инжекции воска** – это специализированное оборудование, предназначенное для точной и дозированной подачи модельного состава в пресс-форму с целью создания восковых моделей, используемых в технологии литья по выплавляемым моделям \(ЛВМ\). Шприц-машины применяются в различных отраслях промышленности, где требуется высокоточное воспроизведение сложных геометрических форм из металла, обеспечивая повторяемость и качество получаемых отливок. Эта машина сочетает в себе универсальность, простоту использования и доступную цену. Идеальное решение для производств с широкой номенклатурой пресс-форм.

**Ключевые преимущества:**
- **Универсальность модельных составов:** Подходит для работы с широким спектром модельных составов, включая парафин-церезиновые и составы на основе синтетических восков.
- **Гибкость управления:** Ручной режим позволяет оператору полностью контролировать процесс впрыска воска, обеспечивая адаптацию к различным типам форм и составов.
- **Высокая точность:** Система контроля температуры воска с точностью ±1°C гарантирует стабильное качество восковых моделей.
- **Подогреваемый бак для воска с перемешиванием:** Предварительная подготовка модельной массы с оптимальной консистенцией и давлением для стабильной подачи.

**Принцип работы:**

Оператор вручную устанавливает и фиксирует пресс-форму. Модельный состав \(воск\) впрыскивается в форму через подвижный рукав инжекции \(впрыск воска производится вне зоны шприц-машины, например, на рабочем столе\).

**Области применения:**
- Производство восковых моделей для точного литья по выплавляемым моделям \(ЛВМ\).
- Изготовление широкой номенклатуры деталей, где требуется быстрая переналадка под различные пресс-формы.
