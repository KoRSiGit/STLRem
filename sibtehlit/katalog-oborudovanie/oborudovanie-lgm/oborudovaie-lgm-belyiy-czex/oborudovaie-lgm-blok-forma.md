# Блок-форма

Блок-форма необходима для спекания полистирольных блоков при помощи термического формирования предвспененного полистирола. Блок-форма состоит из камеры; пульта управления с сенсорным экраном и визуализацией процесcа; гидравлических толкателей для выталкивания блока из камеры; гидравлической двери; инжекторов для подачи предвспененного полистирола в камеру; маслостанции; воздуходувки и системы подвода пара.

Блок-форма изготавливается по заданным заказчиком размерам камеры.
Цикл работы блок-формы следующий:
- Закрытие двери;
- Загрузка предвспененного полистирола в камеру;
- Спекание полистирола;
- Стабилизация и охлаждение блока;
- Открытие двери;
- Выталкивание готового блока

Блок-форма укомплектована программируемым логическим контроллером и цветным сенсорным экраном SIEMENS.
Воздуходувка, входящая в комплект поставки, предназначена для задувки материала, создания вакуума в камере и охлаждения.
