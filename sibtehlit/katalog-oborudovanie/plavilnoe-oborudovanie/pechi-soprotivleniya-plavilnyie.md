# Печи сопротивления плавильные

**Плавильные печи сопротивления** - агрегаты для плавки и выдержки алюминиевых цинковых и некоторых медных сплавов \(до 1200° С\). Конструкция печи может быть стационарной или наклонной, при этом наклон печи осуществляют 2 синхронизированных гидроцилиндра.

Вместо футеровки используется готовый графитовый тигель, легко снимаемые плиты из огнеупорного цемента и кольцо из литого чугуна для быстрой замены тигля. Изоляция из легких огнеупорных блоков, микропористого материала и огнеупорного цемента. Аварийное дренажное отверстие в дне печи для защиты печи в случае повреждения тигля. Нагревательные элементы - резистентная лента из высокотемпературного сплава 0Cr27AL7Mo2. Лента равномерно расположена на внутренней стороне камеры печи при помощи керамических трубок. Блок управления в отдельном шкафу, включая 4 м кабель для соединения с печью. Контроллер НТ 40Т для обеспечения точного контроля температуры. Независимый блок защиты от перегрева. В случае выхода из строя нагревательного элемента он заменяется на новый при извлеченном тигле печи.
По желанию заказчика печь может вместо электрических нагревательных элементов быть укомплектована газовой горелкой для работы на природном газе.
