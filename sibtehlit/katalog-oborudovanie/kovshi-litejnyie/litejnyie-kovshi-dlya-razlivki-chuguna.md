# Литейные ковши для разливки чугуна

**Литейные ковши для разливки чугуна** используются для транспортировки жидкого сплава от [плавильного оборудования](plavilnoe-oborudovanie/index.md "Плавильное оборудование") к месту разливки. Ковши предназначенные для чугуна имеют более широкую горловину для возможности скачивания шлака с поверхности зеркала металла.

Литейные ковши для разливки чугуна серии ТB изготавливаются с соотношением 1/1,1 \(высота ковша в 1,1 раза больше его диаметра у основания\) это позволяет иметь широкое зеркало металла для скачивания шлака вручную. Ковши комплектуются червячным самотормозящимся редуктором с передаточным числом, в зависимости от емкости ковша от 64 до 4820. Данные ковши имеют два сливных носка: для скачивания шлака и для чистого металла. Диаметр маховика для ручного управления наклоном от 600 до 1100мм. Все ковши имеют пояса жесткости, кованые транспортировочные цапфы, поворотные траверсы, червячные редукторы. Ковши поставляются без футеровочных материалов. Рекомендации по футеровке изложены в документации на русском языке, прилагаемой к каждому ковшу. Ковши большой емкости от 10 тонн комплектуются электромеханическим наклоном.

По техническому заданию заказчика ковши могут быть изготовлены с любым соотношением диаметра к высоте, а также с нестандартными размерами поворотной траверсы \(подвески\). Все ковши имеют декларацию о соответствии требованиям таможенного союза.
